Object Position

Esta propiedad sirve para establecer una posición de la imagen. Sus valores pueden ser top, bottom, left, rigth y también se le pueden dar valores en pixeles, rem, em,etc.

Right

Left

Top

Bottom

Para el caso de rigth y left se usa object-fit: cover para que se note mejor la diferencia. Lo que sucede es que la imagen dentro del contenedor se mueve hacia la izquierda o derecha según se indique. Para el caso de top y bottom se utiliza contain y lo que sucede es que la imagen se mueve hacia arriba o abajo según se indique.
