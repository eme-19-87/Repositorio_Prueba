Pasos para subir el proyecto a gitlab:

1-Realizar git init dentro de la carpeta deseada.
2-Realizar git add . para agregar todos los documentos pertinentes
3-Realizar commits: git commit -m mensajeDelCommit.
4-Escribir la siguiente línea git remote add nombreRepositorio URLRepositorio.
    Por ejemplo: git remote add docCSS https://gitlab.com/sietecorrientes/documentacion-html-css.git
5-Realizar la siguiente línea git push nombreRepositorio nombreRama
    Por ejemplo: git push docCSS modificacion29-10-2021
6-Colocar nombre de usuario (usualmente es el mail) y contraseña.
