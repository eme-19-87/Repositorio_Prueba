Normalizar

Los navegadores traen un estilo por defecto. Eso quiere decir que el navegador aplica algunos valores a ciertos atributos para la etiquetas. Por ejemplo, las etiquetas div pueden empezar con margen o padding establecidos. Con la normalización lo que hacemos es reiniciar los atributos del documento para poder trabajar mejor con ellos.
Si haces click aquí puedes ir a la página para descargar el archivo css que ayudará a la normalización.

Archivo

Cuando se haga click en la opción download de la página se abrirá el archivo mostrado en la segunda imagen. En ese archivo se debe hacer click derecho->guardar como y se guarda en la carpeta css del proyecto.

El archivo descargado tendrá ciertos valores que se aplicarán por defecto a todos los elementos html, pero sería conveniente (según el vídeo claro) asegurarnos ciertos valores para determinados elementos:

  • Comprobar en normalize.css que el selector img tenga la propiedad max-width:100%
  • Darle a todos los elementos mediante el selector universal la propiedad box-sizing:border-box.
  • Darle a todos los elementos mediante el selector universal las propiedades padding:0 y margin:0.
