Pseudoclases

Las pseudoclases escuchan un evento. Algunas de las pseudoclases son

  • :hover->sirve para realizar una acción cuando el ratón está encima del elemento.
  • :link->permite darle estilo a un enlace el cual aún no se ha visitado. Sólo se establece el link si la página se visita efectivamente.
  • :visited->permite darle estilo a un enlace el cual ya se ha visitado.
  • :active->permite modificar los estilos cuando se mantiene presionado el botón del ratón sobre el elemento. Si se suelta el botón, el estilo deja de tener efecto y si se suelta durante una transición, la transición no se completa.
  • :focus->se aplica más a los input y se activa cuando el elemento tiene el foco.
  • :lang(lenguaje)->requiere que la etiqueta tenga el atributo lang="nombreIdioma". Cuando hagamos en css objeto:lang(nombreIdioma) podremos darle estilo al texto en el idioma establecido.

Una propiedad interesante que puede usarse con las pseudoclases es la propiedad transiction. Esta propiedad permite establecer un tiempo para la transición desde el valor de una propiedad a otra. Por ejemplo, supongamos el siguiente código: 

            
                .cajaHover{
                    background:hotpink;
                    
                }
                
                .cajaHover:hover{
                    background:khaki;
                    transition: background 0.5s;
                }
Si colocamos a un div la clase cajaHover, el div tendrá un color rosado. Luego, con cajaHover:hover establecemos el cambio de los atributos que sufrirá el div cuando se pase el ratón sobre él. Podemos notar la propiedad transition: background 0.5s nos está indicando que se tome 0.5s para pasar del color rosa al color khaki.
