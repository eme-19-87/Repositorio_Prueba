Float

El atributo float es un atributo que, antiguamente, se utilizaba para ir armando la página mediante bloques. Hoy existen mejores formas de trabajar para posicionar los elementos, por ejemplo mediante grid. Este atributo puede ser utilizado para flotar los elementos a izquierda y derecha. Podemos usar float para lograr efectos similares al mostrado a continuación.

Valores Para Float

Los dos valores que puede tomar float son left y rigth. Cuando se usa rigth se hace flotar el contenido hacia la derecha tratando de acomodarse hacia esa dirección. Con left ocurre lo mismo, pero para la izquierda. Cuando los elementos hacen esto es como si los elementos perdieran la capacidad de seguir el flujo de las etiquetas html. Hoy por hoy, se considera mala práctica usar float. Como los elementos flotan puede haber un problema de superposición con los elementos que no flotan como tal. Veamos con el siguiente ejemlo
